# Daily Practice Log

Add one file per day. I'll analyze your logs and give feedback on patterns, gaps, and progress.

## How to use

1. Copy `_template.md` and rename it to today's date: `2026-06-29.md`
2. Fill it in honestly — takes 5–10 minutes
3. Commit it: `git add practice/ && git commit -m "practice: day X log"`
4. When you want analysis, say: **"analyze my practice logs"**

## What I'll look for when analyzing

- Concepts you keep getting wrong (revisit those teaching files)
- Assessment scores trending up or down
- Time spent vs depth of understanding
- Gaps between what you practiced and what the assessment covered
- Recommendations for what to focus on next

## File naming

```
practice/
  _template.md        ← copy this each day
  2026-06-29.md       ← your first log
  2026-06-30.md
  ...
```
