# Paytm Frontend

## 🎯 Objective
A React-based user interface for the Paytm application, allowing users to sign up, log in, search for other users, and perform money transfers.

## 🛠 Tech Stack
- **Languages:** JavaScript (ES6+)
- **Frameworks/Libraries:** React.js, Tailwind CSS (optional)
- **Tools:** Vite, Axios (presumably)

## 🧠 Key Learnings
- **Component Architecture:** Building reusable UI components for headers, balances, and user lists.
- **State Management:** Handling authentication tokens and user information across pages.
- **API Integration:** Connecting the frontend to the backend using Axios/Fetch for real-time data.

## 🚀 Run Instructions

### Prerequisites
- Node.js v18+

### Setup
1. Navigate to this directory:
   ```bash
   cd frontend/paytm
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Execution
- Run the development server:
   ```bash
   npm run dev
   ```
